SQL Injection

SQL Injection is a format of attack typically used to attack data-driven applications. It's a highly common attack, and allows for users to be able to execute typically malicious SQL statements. These can be used to add, modify, delete, or read records in the database.

How common are SQL Injection attacks?

SQL Injection attacks are one of the most common attacks that are conducted, simply because of the ease and the results that come from having conducted these attacks, poorly secured systems that are holding credit card details can be seen as being an easy target. Whether it be gathering the information that has been found, and selling it on or just using the details that have been found.

How are these conducted?

No expert software needs to be used in order to be conducted, there are like most attacks two different ways that these can be done, either via Intelligence based, or Brute Forced.

These attacks work by inputting SQL code into a user input box, such as a password field and tricking the system potentially allowing for a user to be able to log in, without the need for a password, these attacks can however be used in order to abuse a system, and giving information such as Credit/Debit card numbers, or other details.
