Aircrack Attack (Network Breaching)

Aircrack is a network software suite that is used in to test Wi-Fi network security - however it can be used to allow outside users to gain access to a network. From there an outside user would be able to gain access to a lot of information, as well leaving network users exposed.

How are these conducted?

Using Kali-Linux, an tester/attacker is able to find Networks that are local to them that are able to be penetrated – from there they are then able to manipulate the packets that are being sent from the Wireless Access Point (This is likely to be from a Router, within the home) to make the Access Point think that it’s probing it for Authentication packets so that it’ll send more.

In order to protect from Aircrack, or any other formats of Crack Attacks - the best defense is to be vigilant, and aware of new and developing threats. This advice is prevalent within security.

I just set up my Router is it safe?

Typically out of the box configurations are not safe, and will typically allow for a simple base set up of Security: enough to make sure that other users are not going to be able to get onto your Wireless Access Point unless using other tools in order to enable them to get on there. For example, when someone receives a Virgin Media Super Hub, their standard router - the username and password for the router will be assigned as Admin, with a password of 'Changeme'.

There are services online that offer the passwords to almost all leading brand Routers available for free. Warranting these simple logins and passwords uneffective. Make sure to change the logins for these devices, or chances are that they will be identical to those that are featured on this website.
