Man in the Middle Attack

Aircrack is a network software suite that is used in to test Wi-Fi network security - however it can be used to allow outside users to gain access to a network. From there an outside user would be able to gain access to a lot of information, as well leaving network users exposed.

How are these conducted?

Man in the Middle attacks have changed drastically as new, and more advanced technologies have been developed. Most recently in the news a technique known as TLS Tapping has been widespread across the Internet. The process includes software such as Anti-Virus, Web Development Tools, or Malware which will replace the certificates for different reasons, i.e. Scanning for Malware, Debugging, or just malicious intent respectively.

However not all MITM attacks are conducted in this manner, some include physically being closer to the Network to gain access to the Information that is on said Network. From this, it is highly advised that all users should have high security passwords on all Network Devices, to stop the potential breach, and interception of data on the Network.

How can I protect myself?

There are several different ways to protect yourself from Man in the Middle attacks, these are as follows:

Be Safe
Since you may not know when these attacks are happening, the best preventative measure is to be cautious, don’t use unsecured public Wi-Fi connections, don’t purchase goods on a mobile connection. These connections can be hijacked - only use Wireless connections that you trust such as a protected Home network, or a protected Private network.
Be Secure
Should a HTTPS certificate be untampered, all traffic between you and the server is encrypted. This means that any information that is in the packets cannot be seen through plain text whereas HTTP packets can be read, and private information can be seen (More information on Phishing links here).
Be Vigilant
Make sure that all devices are up to date with latest updates – these will heighten security standards that the device has. Ensure that home networks are protected with high strength passwords – more information about these can be found here. Should it be required, an Intrusion Prevention System can be acquired to make sure that threats cannot breach your Network.

Remember

Understanding how these attacks are composed allow for you to gain a better insight as to how you can protect yourself, and others from these types of attacks.

Still need more assistance?

Did you know that Man in the Middle attacks are prohibited under the Computer Misuse Act 1990, under Section 1 (1B) unless otherwise authorized.
