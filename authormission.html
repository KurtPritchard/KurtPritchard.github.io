Mission Statement

To spread Cyber-Security Knowledge.

To do this there were a number of different factors that were considered:

Accessibility
Being able to extend my reach of this knowledge to everyone. WAVE has been a great tool in ensuring the accessibility of this website.
User-friendly
The information on this website is an introduction towards Cyber-Security. The goal was to allow for users to have a real Understanding on the e-mails they may receive, in their inbox detailing that they've been hacked. This was done by showing users some of the ways how they have been hacked.
Ease of use
Ease of use broadens the amount of users that will stay interested in the website, the user journey was a big factor in making this website. Is it easy to use on Mobiles, and Desktops? Is there anything that could be improved?

Vision Statement

Accessible Cyber Security information, for everyone.

Understanding how these attacks are composed allow for you to gain a better insight as to how you can protect yourself, and others from these types of attacks.
